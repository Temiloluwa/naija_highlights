# Naija Highlights

An NLP project based on weekly nigerian news

## Data

Data is scraped weekly from the following news online platforms
1. https://punchng.com/
2. https://sunnewsonline.com/

## Run crawlers

<br>
Crawl commands:
``` bash
    # to crawl punch
    scrapy crawl punchng

    # to crawl sun
    scrapy crawl sunnewsonline
```
